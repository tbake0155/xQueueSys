Make commands to build and install:

    run "make all" followed by "make install"  // builds xqueuesys, process_tracker and built-in processes

    run "make no_processes" followed by "make install_no_processes"  // builds xqueuesys and process_tracker


Other make options are available, look through the provided make file for more details. 

GTK 2.0 libraries are required to build. You can install them on a debian system with "sudo apt-get install gtk2.0-dev"
